# ⚔️ RPG Guild Management System — Java OOP Assessment

## Overview

In this assessment, you will design and implement a role-playing game (RPG) Guild Management System in Java.

The system models guild members, an item-based inventory system, and a match-based combat system processed through queues.

A partially completed skeleton project is provided. Your task is to complete the implementation by following the TODO instructions in each file while applying Java Object-Oriented Programming principles.

---

## Learning Outcomes

- Object-Oriented Programming (Encapsulation, Inheritance, Abstraction)
- Greenfield Development
- Unit Testing with JUnit
- System Design
- Collections (ArrayList, HashMap)
- Defensive Copying
- Queue-Based Processing
